もうわかるよね
15と表示されます。 ほかにも - (引き算) や * (掛け算) や / (割り算) の四則演算が使えます。
もちろん
int c = a + b * 5 + 10;
みたいなこともできます。
まぁ、表示されるのは45ですね。
で、いま、もし、これで表示されるのが、85と思った方いませんか？
確かに、左から計算していった場合、85になります。
a + b (10 +5)して15
15 * 5 して75
75+10で85
多分その方は小学生だと思われますが...
計算には順番があります。そうですよね。
基本的には左から計算しますが、掛け算、割り算がある場合、そっちを先に計算します。
まあ、見やすいようにすると
a + ( b * 5) +10
ですね。カッコの中を、先に計算するルールがあるので。
で、プログラムでもカッコは使えます。
つまり
int c = (a + b) * 5 + 10;
と、できます。
ちゃんと、85と表示されるはずです。
で、Javaの中で使える演算子と、優先度を表にしてみました。
